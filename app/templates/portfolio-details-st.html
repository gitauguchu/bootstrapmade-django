Style Transfer Machine Learning Project

This is a full stack data science project where I built a web app that is connected to machine learning models that change the style of images passed to it. I developed an asynchronous API with Python and FastAPI, served up the machine learning models using FastAPI, developed a user interface using Streamlit, containerized FastAPI and Streamlit with Docker

Technologies used: FastAPI for the backend to serve our predictions, Streamlit for the user interface, OpenCV for the actual prediction and Docker for containerization
