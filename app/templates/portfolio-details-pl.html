Azure Premier League Data Pipeline

This is a data engineering project where I build a pipeline to handle Extraction, Transformation and Loading. The project involves scrapping the data, cleaning it and staging the data on an azure blob storage. The data is then pushed to a PostgreSQL database from where you can connect to a BI service. The whole process is then automated using GitHub Actions and scheduled to run every Saturday and Sunday.

Technologies used: Python, Azure Cloud Storage, PostgreSQL, GitHub
